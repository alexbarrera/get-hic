Welcome To The GGR Hi-C Data Hub

This site provides an entry point to access the Hi-C datasets associated with the GGR project. The downloaded data are the results from executing the dump command from the Juicebox Command Line Tool program developed in the Aiden Lab. For a detailed description of the output, please see the Juicebox Documentation.

The Data

Tony D'Ippolito in the lab has been investigating the role of chromatin conformation in the Glucocorticoid Response. Blob about Tony's work.

In this site, you can download normalized reads (Knight-Ruiz method) of pair contacts (interaction maps) at different time points after induction of 100nM of DEX (0h, 4h, 8h and 12h of exposure). The output is produced at 5kb resolution and consists in a tab-delimited file in the following format: 

<coordinate1> <coordinate2> <norm_reads>
100000 200000 1102
105000 200000 20.1
...

Accessing The Data

WIP note: in development, only the interaction map after 8hs of DEX is available.

To download the interaction maps, for a given time point, 2 regions of interest (e.g. A and B) must be specified in the following way:

FINAL_URL/<hours after DEX>/<chrA>:<coordA1>-<coordA2>.<chrB>:<coordB1>-<coordB2>
Example:
http://67.159.92.68:32768/projects/GGR/hic/8/chr19:1000000-2000000.chr19:200000-300000
